2. Configuration préliminaire¶

2.1. Brancher le PhotoMiroir¶

Branchez simplement le cordon d’alimentation sur le port micro-USB et le câble de l’appareil photo sur un des quatre ports USB.

Assurez-vous que la carte micro-SD est correctement enfichée.

2.2. Configuration du WIFI¶

Cette section décrit comment vous pouvez configurer le PhotoMiroir pour le connecter à votre réseau WIFI local.

2.2.1. Par le WIFI du boîtier PhotoMiroir¶

Le boîtier du PhotoMiroir met à votre disposition un réseau WIFI dont le nom (SSID) est PhotoMiroir. Une fois le boîtier du PhotoMiroir démarré, vous pouvez y connecter votre ordinateur ou votre smartphone.

Ensuite, vous pouvez utiliser un navigateur web pour naviguer à l’adresse http://photomiroir.fr et cliquer sur le lien admin pour accéder à la page web permettant de changer la configuration du WIFI.

Sur la page de configuration (voir Fig. 2.1), cliquez sur le lien Configuration système et réseaux ❹:

Fig. 2.1 Page d’administration

Et ensuite remplissez les champs host_ssid et host_wifi_pass avec le nom et le mot de passe du réseau wifi auquel vous voulez que le PhotoMiroir se connecte (see Fig. 2.2).

Fig. 2.2 Modifier la configuration du wifi

2.2.2. Par l’intermédiaire de la carte micro-SD¶

  • Insérer la carte micro-SD dans le lecteur de votre ordinateur.

  • Assurez-vous que la petite partition est montée.

  • Créez sur cette partition un fichier nommé wifi.conf. Ce fichier doit décrire la configuration de votre réseau WIFI selon le format suivant:

    network={
   ssid="SSID NAME"
   psk="password string"
}

  • Démontez la carte micro-SD et insérez là dans le boîtier du PhotoMiroir.

  • Branchez le PhotoMiroir. Il doit alors apparaître sur votre réseau.

2.3. Récupérer l’adresse IP locale¶

Une fois le wifi configuré comme décrit dans Configuration du WIFI, vous pouvez récupérer l’adresse IP locale du Photomiroir en vous connectant au wifi ayant le SSID PhotoMiroir puis en naviguant à l’adresse http://photomiroir.fr/admin.

Vous trouverez en haut de la page web quelque-chose comme (LAN IP : 192.168.0.44). L’adresse IP locale est 192.168.0.44. Vous pouvez maintenant vous connecter au Photomiroir en naviguant vers http://192.168.0.44 (remplacez par l’adresse que vous avez vu affichée).
